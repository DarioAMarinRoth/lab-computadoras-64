## Tipos de dato en java

 ### Tipos de dato enteros
 - byte: 8 bits
- short: 16 bits
- int: 32 bits -> preferido
- long: 64 bits

 ### Tipos de dato de punto flotante (racionales)
- float: 32 bits
- double: 64 bits -> preferido

### Otros tipos de dato

-  boolean -> true o false
-  char -> caracteres ASCII

- String (no es un tipo de dato) -> cadenas de texto
